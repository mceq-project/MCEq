Added :class:`~MCEq.geometry.density_profiles.MSIS00LocationCentered`, a general
detector-centred MSIS atmosphere class that computes the shower impact
latitude/longitude from zenith+azimuth angles using full 3-D ECEF geometry, then
feeds those coordinates to NRLMSISE-00.  Includes the
:class:`~MCEq.geometry.density_profiles.MSIS00KM3NeTCentered` convenience subclass
for the KM3NeT ORCA and ARCA sites, a refactored
:class:`~MCEq.geometry.density_profiles.MSIS00IceCubeCentered` that inherits from
the new base class, and the new :meth:`~MCEq.core.MCEqRun.set_zenith_azimuth` API
(deprecating :meth:`~MCEq.core.MCEqRun.set_theta_deg`).
